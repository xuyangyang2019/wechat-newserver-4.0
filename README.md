程序概要说明
---------------------------------

1、建一个名为chat1数据库,编码格式为utf8mb4，MySQL数据库版本建议到8.0以上，不然有些字符低版本的存储不了
2、使用database目录下的脚本初始化表
3、修改application.properties数据库连接信息
4、运行com.jubotech.SpringBootStarter启动服务
5、调试通讯接口略
6、后台http://127.0.0.1:13086/
